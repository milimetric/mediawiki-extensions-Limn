MediaWiki Limn
==============

Zurich Hackathon.  Limn is a MediaWiki extension that uses Vega JS to visualize almost arbitrary data in almost arbitrary ways.

To install:

Add to your LocalSettings.php:

```
require_once("$IP/extensions/Limn/Limn.php");
```
